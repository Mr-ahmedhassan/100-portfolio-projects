# 📊 Income Statement Dashboard — Budget vs Actual Variance Analysis

> **Part of the [99 Projects Portfolio Series](https://github.com/Mr-ahmedhassan)**

## Overview
A management reporting dashboard applying Budget vs Actual Variance Analysis across a full P&L — from Gross Revenue to Net Profit. Each line item shows absolute variance and % deviation from budget, enabling finance teams to identify where performance leads plan and where action is needed.

> 💡 **Budget vs Actual Variance Analysis** compares planned financial targets against real results — line by line — to understand where the business is over- or underperforming. It is a core technique in FP&A and management accounting.
